# Carpeta de Imágenes

Coloca aquí todas las imágenes que necesites para el sitio web.

## Imágenes requeridas:

- **hero-background.jpg** - Imagen de fondo para la sección hero (recomendado: 1920x1080px o mayor)
- Puedes agregar más imágenes según necesites

## Formatos recomendados:
- JPG para fotografías
- PNG para imágenes con transparencia
- WebP para mejor optimización (opcional)
